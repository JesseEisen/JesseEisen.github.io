#Python

阅读simple_dl的代码，有如下的一些收获。

#__all__

在阅读bentoML的simpe_dl的代码时，在__init__.py中通过__all__的方式显示的暴露了模块的相关接口。一个基本的使用是：

....

__all__=[
    "func1",
    "func2",
    "descryptor"
]
  • __all__可以在from xxx imprt *是做一个保护，避免引入不必要的内容。
  • __init__.py在文件夹内表示的是当前文件夹是一个 Python 包，而不是一个普通的文件夹

#Sequence

不同的 python 版本，Sequence 的所在的库也有所不同，需要注意区分。这些只是类型注释，并不会对实际运行有什么干扰。

  • python3.8
from typing import Sequence, TypeVar

T = TypeVar('T')      # Declare type variable

def first(l: Sequence[T]) -> T:   # Generic function
    return l[0]
  • python3.10
from collections.abc import Sequence
from typing import TypeVar

T = TypeVar('T')

def frist(l: Squeence[T]) -> T:
    return l[0]
