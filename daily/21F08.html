记录下 docker 的一些常用命令。

# 查看 container 的详情

$ sudo docker inspect  <container_name/container_id>

# 查看 container 的内存使用情况

$ sudo docker stats

# ubuntu 上查看 docker 日志

$ sudo journalctl -fu docker.service

# 删除 image

$ sudo docker rmi <image-id>

# 删除 container

$ sudo docker rm <container-id/container-name>
